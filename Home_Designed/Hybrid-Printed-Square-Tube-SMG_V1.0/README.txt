# Hybrid 3D Printed Square Section Open-bolt SMG

#### Professor Parabellum

#### Version 1.0

#### Description

An open-bolt, full auto only printable 9mm SMG utilizing an upper receiver made from 30mm x 2mm steel square box section. Compatible with both standard and printable Glock magazines. The trigger and sear block design (Copied from the Minuteman SMG)
can be printed in PLA although a sear block cut from 15mm thick aluminum plate should ideally replace the PLA one.


Two Handguard files are included, one compatible with barrels over 164mm and a shorter version for use with a 114mm long barrel.


#### Parts to print:

		 - Lower Receiver
                 - Magazine catch assembly
		 - M4 Grip
		 - Stock
                 - Handgrip
                 - Trigger
                 - Sear block (Replace with one cut from 15mm thick aluminum or steel plate) 
                 - Rail mount  
                 - Glock magazine body assembly  


#### Print Settings:

		 Recommended Filament: eSun PLA+
		 
		 - Nozzle Temp             - 230C
		 - Bed Temp                - 60C
                 - Infill                  - 100%
		 - Brim                    - On
		 - Supports                - None
		 - Layer Height            - 0.15mm
		 - Prep                    - Print on a glass bed prepped with a gluestick
