# KRISS Vector - Flush accessory cover mod

## About
Made in Fusion 360. Parts to replace the picatinny rail on the Vector pistol.

Parts include:

- Flush Handstop
- Flush Handstop, with angled foregrip
- Base Cover - the base model for the handstops. (Use this to create new designs!)
