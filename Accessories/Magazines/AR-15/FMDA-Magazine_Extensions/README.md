# FreeMenDontAsk Presents
## Printable Magazine Extensions for a Variety of AR15 Mags

### What you'll need:

     - PLA (PETG is also tested as working)
     - A printer capable of printing PLA (or the plastic you choose to use).  The Creality Ender 3 is capable.
     - An appropriate 30 round AR15 mag spring
	  
### Print Settings
	 
         - Print mag extensions upright, no supports.
	
	 - Print Settings (for PLA)
		 
		 - Nozzle Temp             - 230C
		 - Bed Temp                - 60C
		 - Brim                    - ON
		 - Supports                - NONE
		 - Layer Height            - 0.15mm
		 - Bed prep                - Gluestick, optionally on top of Kapton Tape.

### Understanding the files
	- Extension for Hex Mag is +10
	- Extension for Korean Mag is +5
	- Extension for Lancer Mag is +30 and +10
